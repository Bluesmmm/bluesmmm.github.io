# Jeff Liu's Tech Blog

Welcome to my personal tech blog! This site is built with [Hexo](https://hexo.io/) and the [NexT](https://theme-next.js.org/) theme.

## Live Site

Visit the blog at: **https://bluesmmm.github.io/**

## Quick Start

### Install Dependencies

```bash
npm install
```

### Local Development

```bash
hexo server
```

Open http://localhost:4000 in your browser.

### Create a New Post

```bash
hexo new "Post Title"
```

### Build & Deploy

```bash
hexo clean && hexo generate && hexo deploy
```

## Tech Stack

- **Hexo** - Fast static site generator
- **NexT Theme** - Clean, elegant theme with dark mode
- **Markdown** - Content authoring
- **GitHub Pages** - Free hosting

## Project Structure

```

│   └── _data/
```
